# Game of Life
A simple version of the Game of Life

## Usage

	Input format

	L C S 		L - (rows) | C - (cols) | S - (steps) are Integers >= 1
	row_1
	row_2		Each row is a sequence with C characters
	……………		Characters can be '#' or '.'
	row_L			


	Output Format
	
	row_1
	row_2
	……………
	row_L
